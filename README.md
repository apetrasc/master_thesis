今後、マニュアルファイル（.md）はこのディレクトリに配置してください。

## 使用方法

LaTeX Workshopを使用してPDFを生成・プレビューします。

### PDFの生成方法

1. `main.tex`を開く
2. `Ctrl+Alt+B` (Windows/Linux) または `Cmd+Option+B` (macOS) でビルド
3. または、LaTeX Workshopパネルから「Build LaTeX project」ボタンをクリック

### PDFのプレビュー方法

1. `Ctrl+Alt+V` (Windows/Linux) または `Cmd+Option+V` (macOS)
2. または、LaTeX Workshopパネルから「View LaTeX PDF」ボタンをクリック

## フォント設定

フォントは`ptex-ipaex.map`を使用しています。このファイルはシステムにインストールされているTeX Liveのフォントマップファイルです。
